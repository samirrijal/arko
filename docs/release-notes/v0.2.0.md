## What's new since v0.0.1

### Method presets — named slate 4/4 + AR5 bonus

`MethodRegistry::standard()` now ships **5 presets**:

- **IPCC AR6 GWP100** (`ipcc-ar6-gwp100@1`) — recommended default; AR6
  WG1 Ch7 T7.15; CH4 with `FlowOrigin × {Fossil, Biogenic, LandUseChange}`
  splits + `LandUseChange→Fossil` equivalence wiring.
- **IPCC AR5 GWP100** (`ipcc-ar5-gwp100@1`) — legacy-parity bonus for
  historical EPDs (with climate-carbon feedback).
- **EF 3.1** (`ef-3.1@1`) — first non-climate preset; 7 EN 15804+A2
  emission-core indicators (CC, OD, POCP, AC, EU-fw, EU-m, EU-t). Per
  [`D-0015`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md).
- **CML-IA baseline 4.8** (`cml-ia-baseline@4.8`) — Leiden CML, August
  2016, EN 15804+A2-aligned subset (7 categories). Satisfies the
  Execution Guide's "CML 2001" slate via Leiden continuation lineage.
  Per [`D-0017`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md).
- **ReCiPe 2016 Midpoint Hierarchist** (`recipe-2016-midpoint-h@1.1`) —
  RIVM v1.1 (post-erratum); 10 categories (7 EN 15804-aligned + 3
  ReCiPe-distinctive: PMFP, land occupation, water consumption);
  GLO-only matchers. Country-specific CFs + `CasRegion` + per-process
  Eq. 3 restructure deferred to V2 as a coupled regionalisation
  bundle. Per [`D-0019`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md).

Three new I/O crates extend the workspace from 13 → 16:

- **`arko-io-ilcd-linker`** — multi-file ILCD bundle linker. Reads
  ÖKOBAUDAT, JRC EF reference packages (94k flows resolved). Parses
  `FlowOrigin` from basename parenthetical; closed a pre-existing

- **`arko-io-lcax`** — LCAx v3.4 writer (V1). Closes the Phase-1

  LCAx reader and ILCD+EPD writer staged for Phase 2.
- **`arko-io-olca-jsonld`** — openLCA JSON-LD reader. Reads USDA LCA
  Commons (CC0 1.0). Per [`D-0014`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md).

### Taxonomy extensions

- **`FactorMatch::CasCompartment`** added per [`D-0015`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md):
  EF 3.1 acidification counts SO2 to air, not water; eutrophication
  splits N species by freshwater/marine/terrestrial end compartment.
  Brings `FactorMatch` to 5 variants.
- **`FlowOrigin::LandUseChange`** added per [`D-0016`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md):
  EF 3.1 groups LULUC CH4 with fossil at 29.8, not biogenic at 27.0.
  `NonFossil` renamed to `Biogenic` in the same change. Brings
  `FlowOrigin` to 4 variants.

### `FactoredSolver` trait

New surface in `arko-solvers-dense` for factor-once-solve-many
workflows (sensitivity sweeps, Monte Carlo, the §10 incremental
recalc path):

```rust
pub trait FactoredSolver {
    type Factorization;
    fn factorize(&self, a: &SparseMatrix) -> Result<Self::Factorization, EngineError>;
}

pub struct DenseLuFactorization { /* opaque */ }
impl DenseLuFactorization {
    pub fn solve(&self, f: &SparseVector) -> Result<DenseVector, EngineError>;
    pub fn dim(&self) -> usize;
}
```

`Solver::solve` refactored to delegate: `solve(A, f) ≡ factorize(A)?.solve(f)`.
Both surfaces share one numerical kernel; future drift between them
is mechanically impossible.

## What's deferred to V2 (with reasoning)

Each deferral has an active decision-log entry and a concrete trigger
condition — none are vague TODOs.

- **Regionalisation bundle** ([`D-0019`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md)) —
  `FactorMatch::CasRegion` + per-process `C` build + Eq. 3 restructure
  (`h = Σₚ Cₚ · B[:,p] · s[p]`) + 5 country-specific CF tables for
  ReCiPe + reader region-extraction verification + `ProcessMeta.geography`
  promotion. Coupled bundle; ships when a real user requests region-aware
  LCA.
- **USEtox toxicity slate** ([`D-0017`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md)/[`D-0019`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md)) —
  5 categories spanning EF 3.1, CML-IA, and ReCiPe; needs paired
  cross-witnessed factor-table-entry pass.
- **LCAx reader + ILCD+EPD writer** ([`D-0018`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md)) —
  Phase 2; ILCD+EPD is the format program operators (Environdec, IBU,
  EPD International) actually accept.
- **Egalitarian / Individualist ReCiPe perspectives** ([`D-0019`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md)) —
  same matcher infrastructure, different factor tables. Pure
  factor-table addition; ship if user demand surfaces.
- **CML-IA toxicity, regional variants, POCP low-NOx** ([`D-0017`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md))
- **EF 3.1 water use + additional EN 15804+A2 indicators** ([`D-0015`](https://github.com/samirrijal/arko/blob/v0.2.0/DECISIONS.md))

  not factor entry; `engine/core/src/license.rs` is the V1 hook).
