# drafts/blog-posts/

Seed ideas for future Arko writing. **Not published** — files here are
thinking-in-progress, not publishable content. Promotion to the Arko
blog (or any public surface) is an explicit step that happens when the
piece is finished, fact-checked, and the moment is right.

One file per topic. Each file carries:

- The core observation or argument in a paragraph.
- The evidence or experience it's grounded in.
- The intended audience.
- Open questions the draft still needs to answer.

When a draft is promoted, it moves from here to the published blog
surface and this file is either deleted or replaced with a one-line
pointer to where it landed.
